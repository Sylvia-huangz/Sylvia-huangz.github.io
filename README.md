# Sylvia-huangz.github.io

黄子越个人主页（Portfolio Resume）  
基于静态页面构建，包含个人概览、教育背景、关键经历、语言能力与工具栈展示。

## 文件结构

- `index.html`：主页结构与内容
- `styles.css`：页面视觉样式与响应式布局
- `script.js`：滚动显现与数字动画

## 本地预览

直接双击打开 `index.html`，或使用任意静态服务器预览。

## 部署

本仓库为 GitHub Pages 用户主页仓库（`<username>.github.io`），推送到 `main` 分支后会自动生效。
